Clinical Context

Congenital surgery is categorically different. So is the model.

THE PATIENT POPULATION
  • Neonates, infants, children — some days old
  • Single-ventricle, biventricular, complex structural defects
  • Adult risk models do not apply
  • Procedure complexity varies enormously — ASO to Norwood
  • Registry data from AQAP (AATS Quality Assurance Program)
THREE VALIDATED MODELS
BIV Model (47 variables): ASO · ASOVSD · CoAVSD · Truncus · Biventricular repairs
Norwood Model (21 variables): Stage 1 single-ventricle palliation
Hybrid Model (15 variables): Hybrid palliation — ductal stenting + banding
MODEL ARCHITECTURE
All three models use randomForestSRC (R) trained on AQAP registry data. The API automatically routes to the correct model based on the operation field. Only one field is required — all others default to lowest-risk values.
Output
Perioperative mortality % + percentile rank
Percentile
Risk rank vs all historical cases for that procedure
API
Plumber R · JWT auth · Swagger UI
Defaults
Missing fields → lowest-risk values (safe for partial data)
A percentile of 85 means the patient's predicted risk is higher than 85% of historical cases for that same procedure — powerful for family counseling and consent.

SMART DEFAULTS
Only operation is required. All other variables default to lowest-risk: binary fields → "no", procedure-specific factors → "missing", age → 10 days. Enables rapid triage with partial data.
KEY VARIABLE CATEGORIES
PatientAge (days), weight (kg), gestational age (weeks), sex
AnatomySingle ventricle, arch obstruction, heterotaxy, aortic arch anomalies
AcuityShock, mechanical support, cardiogenic compromise pre-op
ComorbiditiesChromosomal anomalies, renal, pulmonary, neurological
ProcedureProcedure-specific factors (PSF), prior cardiac surgery, urgency
InstitutionHospital volume, center experience, bypass time
